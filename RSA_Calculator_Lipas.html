Choose two distinct primes p and q

Pick two distinct prime numbers. n = p·q is the modulus. Small primes work fine for textbook examples.

Modulus n and totient r = φ(n)
Choose public exponent e

e must satisfy 1 < e < r and gcd(e, r) = 1. d is computed automatically as the modular inverse of e mod r.

Each letter is encoded as a 2-digit number (A=00 … Z=25), then: c = me mod n.

Sign a message — sender's private key (n, d)

The sender signs using their private key: S = md mod n. The recipient verifies with the sender's public key.

Verify a signature — sender's public key (n, e)

The recipient recovers: m = Se mod n. If it matches the original message, the signature is valid.

Algorithm summary
Key generation
1. Choose two distinct primes p, q.
2. Compute n = p·q  and  r = φ(n) = (p−1)(q−1).
3. Choose e:  1 < e < r,  gcd(e, r) = 1.
4. Compute d = e−1 mod r  so that e·d ≡ 1 (mod r).
5. Public key: (n, e)  ·  Private key: (n, d).
Encryption / Decryption
Encode each letter: A=00, B=01, … Z=25. One letter per block.
Encrypt:  c = me mod n
Decrypt:  m = cd mod n
Digital signature
Sign:    S = md mod n  (sender's private key)
Verify:  m = Se mod n  (sender's public key)
Valid if recovered m matches the original message.
